### 演示要点提示

1. **数据准备**：确保后端已启动（`cd backend && flask run --port 5000`），前端已启动（`cd frontend && npm run dev`），数据库已 seed（`cd backend && python seed_data.py`）
2. **高德地图 Key**：确保 `.env` 中配置了有效的 AMAP_KEY，否则地图组件可能无法加载
3. **录制建议**：使用 OBS Studio 录制，分辨率 1920x1080，帧率 30fps 即可。录制后可用剪映添加字幕和转场
4. **口播配合**：每个页面切换时留 1-2 秒空白方便后期剪辑
5. **超时控制**：若某功能加载较慢，录制时可先等待加载完成再继续操作，后期可加速片段
